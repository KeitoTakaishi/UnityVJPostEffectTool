## Description
This is PostEffect package(HLSL).
### Containts
- InvertColor
- Zoom
- RGBShift
- Glitch
- GridFlash
- VerticalSymmetry
- HorizontalSymmetry
- Mosaic
- Tile
## Usage

1. PostEffectApply.cs attach MainCamera
2. Make IsEffectSwitch(PostEffectApply's property) true to change effect automatically
